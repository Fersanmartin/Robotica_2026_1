# Robotica_2026_1
Repositorio para tutorial mujoco y trabajos simulados


## Requerimientos:
Python 3.9 - 3.12

## Instalación

### Opción 1  
Ocupar git para obtener repo
```bash
git clone https://github.com/Fersanmartin/Robotica_2026_1.git
cd Robotica_2026_1
```
### Opción 2 
Decargar zip y descomprimir carpeta

### Crear espacio de trabajo
En la carpeta Robotica_2026_1

```bash
py -m venv venv

```
### Descargar dependencias
```bash
venv\Scripts\activate
pip install -r requeriments.txt
```

### Probar codigo
```bash
python
```
